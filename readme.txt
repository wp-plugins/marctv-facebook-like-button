== Description ==

Appends the Facebook "Like-Button" to every .single .entry dom element - after the user scrolled the page.

== Installation ==

* Upload the plugin to your blog
* Make sure that your theme features a canonical tag on every page and that you use the standard classes for you markup. e.g. "".single .post
* Activate it.
